### What I learned

Center element on screen with css:
```css 
.body {
    width: 100vw;   
    height: 100vh;
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
}  
```

```css 
footer {
    position: absolute;
    bottom: 0;
}
```
